Cache Tushare Pro Api data to LevelDB


* Free software: MIT license
* Documentation: https://tusharedb.readthedocs.io.


Features
--------

* 一键同步历史
* 支持增量同步数据
* 支持命令行 tusharedb --help
* 支持批量处理数据

[由于tushare pro 增加了调用限制，1分钟只能发送150请求....]
